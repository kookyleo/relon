# What is Relon?

Relon is a **programmable configuration language** designed specifically for modern web frameworks (such as UI component libraries) and complex industrial-grade configuration systems.

You can think of Relon as a superset of JSON with full expression capabilities. It features complete type inference and structural contracts akin to TypeScript, but deliberately strips away all complex side-effects and statement-based control flows.
